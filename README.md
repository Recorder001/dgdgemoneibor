# 둥글둥글 감정마을 🌿

나를 찾아가는 힐링 비주얼 노벨 게임

---

## 🎮 실행 방법

1. `dungledung-maul` 폴더 안의 **`index.html`** 파일을 더블클릭하거나 브라우저로 열기
2. 게임 시작 화면에서 **Gemini API 키**와 **닉네임** 입력
3. **마을로 출발하기** 버튼 클릭 → 여행 시작!

> **권장 브라우저**: Chrome, Edge, Firefox 최신 버전  
> 인터넷 연결이 필요합니다 (Gemini API 호출)

---

## 🔑 Gemini API 키 발급 방법

1. [Google AI Studio](https://aistudio.google.com/app/apikey) 접속
2. Google 계정으로 로그인
3. **"Create API Key"** 클릭
4. 생성된 키(`AIzaSy...` 형태)를 복사
5. 게임 시작 화면에 붙여넣기

> API 키는 무료로 발급되며, 게임에서 브라우저 localStorage에만 저장됩니다.

## 📁 폴더 구조

```
dungledung-maul/
├── index.html    ← 게임 시작점 (브라우저로 열기)
├── style.css     ← 전체 스타일 (캐릭터 CSS, 애니메이션 등)
├── game.js       ← 게임 핵심 로직 (상태 관리, 화면 전환 등)
├── api.js        ← Gemini API 연동 (응답 분석, 편지 생성)
├── data.js       ← 게임 데이터 (챕터, NPC, 대사, 성격 키워드)
└── ending.js     ← 엔딩 시스템 (편지 4통, 최종 결과)
```

## 🌟 게임 소개

### 4가지 성분 지수
| 성분 | 의미 |
|------|------|
| **O / I** | 마음의 방향 (함께하는 즐거움 vs 나만의 평온함) |
| **D / V** | 세상을 보는 눈 (세심한 관찰 vs 넓은 상상) |
| **H / R** | 결정하는 법 (따뜻한 공감 vs 바른 판단) |
| **S / F** | 행동의 박자 (차근차근 계획 vs 자유로운 흐름) |

### 4개 챕터
- **Chapter 1 광장** — 쿼카 & 고슴도치와 함께 (O vs I)
- **Chapter 2 언덕** — 비버 & 아기 양과 함께 (D vs V)
- **Chapter 3 찻집** — 리트리버 마루 & 올빼미 선생님과 함께 (H vs R)
- **Chapter 4 기차역** — 너구리 역장 & 다람쥐 여행가와 함께 (S vs F)

### 16가지 성격 유형
마지막엔 나만의 고유한 성격 키워드와 친구들의 편지 4통이 기다려요 💌
